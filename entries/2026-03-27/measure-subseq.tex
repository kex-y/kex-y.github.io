Here's a fun measure theory puzzle I came up with and I'm keeping track of it here in case I'm ever 
asked to come up with measure theory exam questions :)

For any sequence of \((x_n)_{n \in \mathbb{N}}\) with \(x_n \in \{0, 1, \dots, 9\}\) for any \(n \in \mathbb{N}\), 
we define the set 
\[S((x_n)) := \{0.a_1a_2a_3\ldots \in [0, 1] : (a_n) \text{ is a subsequence of } (x_n)\}.\]
Then, does there exists a sequence \((x_n)\) such that \(S((x_n))\) has non-trivial measure (i.e. not zero nor one)?

Of course, this question is only meaningful if the sets \(S((x_n))\) are measurable. So a complete 
answer will need to consider this also.
